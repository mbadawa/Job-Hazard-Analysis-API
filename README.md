# How to run the API

### Make sure you have the following installed:
##### Node, MySQL Server, and MySQL Workbench


### How to start up the backend

##### 1 - Open MySQL Workbench and create a database called JHAReport
##### 2 - Run npm install
##### 3 - Run npm run server
